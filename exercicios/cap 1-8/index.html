Usando o site Favicon.io vc consegue gerar a partir de um Png

Favicon é o icone que aparece na aba do navegador, para adicionar, basta colocar o seguinte código dentro da tag head:

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

Negrito / Destaque

Para deixar o texto em itálico, basta usar a tag <i> ou <em>
